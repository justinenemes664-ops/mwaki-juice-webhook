# Mwaki Juice Point WhatsApp Webhook

Webhook server for WhatsApp Cloud API.

## Features

- WhatsApp Webhook Verification
- Receives incoming messages
- Ready for Meta WhatsApp Cloud API
- Deployable on Render

## Environment Variables

PORT=10000
VERIFY_TOKEN=mwaki2026

## Start

```bash
npm install
npm start
```# mwaki-juice-webhook
